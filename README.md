# Alasuma
مصنع فحم العاصمة

## بنية المشروع
- `index.php` — صفحة الموقع العامة
- `admin.php` — لوحة تحكم الأدمن منفصلة
- `api.php` — واجهة برمجة التطبيقات لمعالجة تسجيل الدخول والمقالات والمعرض والمنتجات
- `db.php` — إعدادات اتصال MySQL عبر PDO
- `schema.sql` — مخطط قاعدة البيانات وجدول المشرف
- `assets/css/style.css` — أنماط الموقع المشتركة
- `assets/js/app.js` — سكريبت الواجهة العامة ولوحة التحكم
- `install.php` — مثبت تلقائي ينشئ قاعدة البيانات والجداول وحساب الأدمن

## تشغيل المشروع
1. عدل `db.php` إذا كان اسم المستخدم أو كلمة المرور أو اسم قاعدة البيانات مختلفين.
2. افتح `install.php` في المتصفح لتثبيت قاعدة البيانات تلقائياً وإنشاء حساب الأدمن.
3. بعد التثبيت، افتح `index.php` للموقع العام و`admin.php` للوحة التحكم.
